Exercise 4: Bubble sort

We can also compare strings to other strings. The comparison for strings is done numerically (by comparing the ASCII values of each letter). The strings are compared letter-by-letter, from left-to-right. The comparison continues as long as the characters from the two strings are identical; as soon as different characters are found, the string that has the character with the smaller ASCII code comes before the other string.

Lists can be compared to other lists, and tuples to tuples. if you compare two lists/tuples in Python, they will be considered to be equal if both have the same number of elements, and each element of the first list is equal to the element at the same position in the second. If they have different lengths, the longer one is considered greater.

The code below implements a very simple (but inefficient) algorithm called bubble sort. It sorts a list into ascending order. The algorithm works by comparing adjacent items in the list. If they are in the wrong order, swap them. At the end of the first iteration of the outer loop, the smallest item in the list has "bubbled" up to the front of the list. This process repeats for the second smallest, third smallest, etc. until the list is sorted.

for xx in range(len(values)-1): for yy in range(xx+1, len(values)): if values[xx] > values[yy]: values[xx], values[yy] = values[yy], values[xx]
