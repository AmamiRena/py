Exercise 1: Generating 2D Arrays

Be sure to import numpy

There are four ways we will generate 2D arrays from scratch, using the following four functions:

np.random.random(tuple) np.zeros(tuple) np.ones(tuple) np.reshape(vector, tuple)

The function np.zeros(tuple) creates a 2-D array of floats with value 0.0, according to the dimensions in the tuple. In this lab, our tuples will be 2-tuples of integers, like (10,10). The functions np.ones and np.random.random work in the same way.

We can also use np.reshape. Here, we provide a 1D array containing the elements, and it is reshaped into a 2D array.

Try the following examples.

x1 = np.random.random((5,5)) x2 = np.zeros((10,10),dtype=int) x3 = np.reshape(np.arange(1,13,dtype=int), (4,3))

Exercise 2: Indexing 2D Arrays

To index a 2D array, use a tuple with two integer indexes. For example, if A is a 3x5 array, then A[2,4] is a valid index for the array, but A[4,2] is not. A.shape will show the dimensions of any numpy array.

Try the following examples.

nums = np.reshape(np.arange(1,13,dtype=int), (4,3)) print ("nums:\n{0}".format(nums)) print (nums.shape) print ( nums[3,2]) print (nums[2,3])

Exercise 3: Slices of 2D Arrays

2D Arrays can be sliced in the same way that arrays can, except that both dimensions need to be specified.

Try the following examples.

nums = np.reshape(np.arange(1,13,dtype=int), (4,3)) nums2 = nums[1:3,1:3] # slice the array. nums[1:3,1:3] = np.zeros(nums2.shape) # set to zero

Slices can also include a single index, to slice a row or column from an array.
