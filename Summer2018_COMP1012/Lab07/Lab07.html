Exercise 2: Function with parameters

Write a function that will take two values as its parameter and calculate the product of the two values. The result should be printed from inside the function. So the function SHOULD NOT RETURN anything

Declare a function with name multiplication with two parameters a and b. The function definition should look like this: def multiplication(a,b). Calculate the multiplication of a and b inside the function and also print the result from inside the function.

Exercise 3: Function with parameters and return values

Write a function that will take two values as its parameter and calculate the product of the two values. The result should be printed from outside the function. So the function SHOULD RETURN the multiplication value between the numbers.

Declare a function with name multiplication2 with two parameters a and b. The function definition should look like this: def multiplication2(a,b). Calculate the multiplication of a and b inside the function and return the value from where it is called. Save the returned value in a variable and print the value.
