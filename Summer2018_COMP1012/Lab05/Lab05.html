Exercise 1: Tuples.

  1. Create two tuples representing points in a plane— A:(3.5,1.2) and B:(-2.7,-3.2). Print them. What format code should you use to print a tuple? A simple way to print a tuple is to convert it to a string representation using the str function, then print it using a string placeholder.
  2. Create another tuple representing a vector from A to B (that is, B-A). Notice you can't just subtract one from the other; tuples don't support subtraction. Perform the subtraction one coordinate at a time by subscripting in the original tuples. Print the vector.
  3. Create another tuple representing a point C:(4.2, 2.8), and a tuple for the vector C-B. Print the points and vector.
  4. Calculate and print the sum of the two difference vectors (C-B) + (B-A) by adding their corresponding elements. Calculate and print the vector difference C-A. Are these two the same? Test their equality using the == operator.

Exercise 2: List

  1. Write a program that will take 10 inputs from the user and append the inputs to a list. After completing taking inputs print each value in the list. For taking inputs you need to use for loop. The inputs can be a number or a string. Remember to use a loop for printing the value because if you directly print the list, it will print bracket at the beginning and at the end.
  2. Create a list with 5 integers and find the sum of all elements in the list. To find the sum, you don't need to use any loop. You can use the sum function. for example, if _x is your list, then sum(_x) will give you the sum of the list.
  3. Create another list with 5 integers. Subtract the corresponding element from the list you created in (2). Add the subtraction result to another list and print the results.
  4. Convert the range(1,11) to a list.
  5. Find first n odd numbers to a list where n is the input from the user. Add the numbers to a list and then print the numbers.
  6. Write a program that will ask the user to give inputs of 5 coordinates. Each coordinate contains two values (x-axis, y-axis). Use a for loop to take 5 inputs from the user. Inside for loop take two inputs: one for x coordinate and another for y coordinate and then create a tuple with those inputs. After creating the tuple, add the tuple to a list and print the tuples.

Exercise 3: Sets

  1. Create two sets with 1, 5, 3, 7, 3, 5, 9, 2 and 2, 4, 7, 0, 6, 9
  2. Print the sets. Are there any duplicate elements in the set?
  3. Find the union and intersection of the sets and print the results. You need to use a loop to print the results to avoid {} to show up in the print. But in the set, you cannot directly use an index to invoke an element in the set. First, you need to convert the set to a list and then you can print it.
  4. Find the set difference and again print the result.

Exercise 4: Sets

Create a dictionary that contains entries for the sports teams for Canadian cities. The key values are the names of the cities, and the data item for each city is a list of the sports teams in that city. To get you started, you can copy and paste this string into your script:

{'Winnipeg': ['Jets', 'Blue Bombers', 'Goldeyes' ,'Moose'], 'Vancouver': ['Canucks', 'Lions', 'Whitecaps'], 'Hamilton': ['Tiger Cats', 'Cardinals']}

Add at least one more city to your dictionary. Run your script. Now try accessing the dictionary in the IPython console to find the names of the teams in a particular city, for example if your dictionary is called teams, then typing teams['Winnipeg'] will list the value for that entry.
