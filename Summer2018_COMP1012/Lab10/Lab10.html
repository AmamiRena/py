Exercise 1: The Graphics backend

matplotlib plots are displayed using various tools. It is useful to know the benefits of these tools, and how to choose them. The Graphics backend specifies how plots are displayed.

To change the Graphics backend:

  • Open the Preferences dialog:
    • In Windows, it is under the Tools menu
    • In MacOS, it is under the Spyder menu
  • In the left pane, choose IPython console
  • In the right pane, choose Graphics
  • Select an option from the Backend list:
    • Inline; the plot appears in the console itself (quick, but not adjustable or saveable)
    • Automatic; the plot appears in a separate window (can resize the plot and save as a png image, but cannot see multiple plots)

You may need to close the current IPython console for the change to take effect. You will use the Automatic backend for exercises 2 and 4, and the Inline backend for exercise 3.
